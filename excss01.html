HTML에서 CSS를 사용하여 엘리먼트의 스타일을 정의하는 방법 1. 엘리먼트의 style 속성의 속성값으로 지정 (비권장) "스타일속성명:스타일속성값;스타일속성명:스타일속성값" 형식으로 스타일 속성값 설정 2. head 내에 style 엘리먼트를 추가하여 스타일 지정 선택자 { 스타일속성명 : 스타일속성값; 스타일속성명 : 스타일속성값; } 3. 별도의 파일에 css로 스타일을 정의하고, head 내에 link href="CSS파일명" rel="stylesheet" type="text/css" 엘리먼트로 css 파일을 문서에 포함 (권장) 4. css 파일 내부 또는 style 엘리먼트 내부에서 @import url("CSS파일명"); 으로 다른 CSS 파일을 포함 가능
